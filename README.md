# Merizo Landing Page

This folder is a standalone copy of the Merizo landing page. It does not need the Expo app, backend, MongoDB, or any install step.

Run locally from this folder:

```sh
python -m http.server 5173
```

Then open:

```txt
http://127.0.0.1:5173
```

The CTA buttons open Google Play on Android, Apple App Store search on iOS, and keep the web app behavior by navigating to `/login` on desktop.
